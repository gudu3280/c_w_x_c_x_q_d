# 🐾 宠物服务微信小程序

基于 **uni-app (Vue 3)** + **Spring Boot** 的全栈宠物服务小程序，覆盖宠物寄养预约、美容预约、宠物商城、宠物档案管理等核心场景。

> 🎨 UI 主题采用星露谷物语像素风设计

## 📱 功能模块

| 模块 | 功能 |
|------|------|
| 🏠 首页 | Banner轮播、服务入口、附近商家推荐（基于真实GPS定位） |
| 🔍 搜索 | 500ms 防抖搜索商家 & 商品 |
| 🗺️ 附近服务 | 列表/地图双模式、按寄养/美容筛选、距离排序 |
| 📋 商家详情 | 封面/评分/评价、📅 可预约日历、💰 寄养价格表、✂️ 美容服务 |
| 📝 预约服务 | 宠物健康信息卡片（💉疫苗/🏥病史）、动态费用计算 |
| 🐾 宠物档案 | CRUD、照片上传、疫苗记录、病史/过敏史、生活习惯 |
| 🛒 宠物商城 | 分类浏览、商品详情、用户评价、购物车、地址管理 |
| ⭐ 评价系统 | 订单完成后5星评价、500字内容 |
| 📋 我的订单 | 预约/商城双Tab、状态筛选、取消/评价 |

## 🛠 技术栈

- **框架**: uni-app (Vue 3 Composition API `<script setup>`)
- **状态管理**: Pinia
- **样式**: SCSS + 像素风组件体系
- **定位**: 微信 `wx.getLocation` + gcj02 坐标系，降级至北京默认坐标

## 🚀 本地运行

1. 用 HBuilder X 打开项目目录
2. 配置后端地址 → 修改 `utils/request.js` 中的 `BASE_URL`
3. 点击 **运行 → 运行到小程序模拟器 → 微信开发者工具**
4. 微信开发者工具中勾选 **不校验合法域名**

> 确保后端 Spring Boot 服务已在 `localhost:9999` 启动
